## 测试说明

请检查以下内容：

1. ✓ 引用块的背景色是否合适（墨水屏友好）
2. ✓ 左边框或分隔线是否清晰
3. ✓ 嵌套引用的层级是否分明
4. ✓ 引用内各元素的渲染是否正确
5. ✓ 中英文混排的引用显示是否正常
6. ✓ 长文本引用的换行是否合理
7. ✓ 引用块与普通文本的间距是否适当
