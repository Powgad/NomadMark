## 测试说明

请检查以下内容：

1. ✓ 表格边框是否清晰（墨水屏友好）
2. ✓ 表头是否突出显示
3. ✓ 对齐方式是否正确（左、中、右）
4. ✓ 单元格内的内容是否完整显示
5. ✓ 长文本是否正确换行
6. ✓ 表格中的格式化（粗体、代码、链接）是否正常
7. ✓ 中文字符显示是否正常
8. ✓ 数字对齐是否整齐
9. ✓ 空单元格的处理是否正确
10. ✓ 墨水屏显示效果（对比度、可读性）
