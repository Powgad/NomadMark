# NomadMark Testing Checklist
## Supernote A6 X2 Nomad 真机测试清单

---

## Pre-Installation (安装前准备)

### Device Setup
- [ ] Supernote A6 X2 Nomad 已连接到开发机器
- [ ] 开启开发者选项
- [ ] 开启 USB 调试
- [ ] 已安装 Nomad 版本 1.8 或更高

### Environment
- [ ] Android SDK Platform Tools installed
- [ ] ADB (Android Debug Bridge) 可用
- [ ] Rust Core 已编译 (libmarkdown_core.so)

---

## Installation (安装)

### APK Installation
```bash
# 方法 1: 通过 ADB
adb install -r android/app/build/outputs/apk/debug/app-debug.apk

# 方法 2: 通过 Supernote 内置文件管理器
# 将 APK 复制到设备，然后使用文件管理器安装
```

- [ ] APK 安装成功
- [ ] 应用图标出现在主屏幕

## Functional Tests (功能测试)

### 1. 文件操作
- [ ] 创建新文件
- [ ] 打开现有 Markdown 文件
- [ ] 保存文件
- [ ] 文件内容正确保存

### 2. 编辑功能
- [ ] 外接键盘输入正常
- [ ] 软键盘输入正常
- [ ] 光标移动正常
- [ ] 文字选择和删除正常

### 3. 预览模式
- [ ] 切换到预览模式
- [ ] Markdown 渲染正确
- [ ] 切换回编辑模式

### 4. 分屏模式
- [ ] 外接键盘自动切换到 1:1 分屏
- [ ] 软键盘自动切换到 4:6 分屏
- [ ] 编辑区和预览区显示正确

### 5. 修订模式
- [ ] 进入修订模式
- [ ] Ratta 手写笔识别正常
- [ ] 手势删除/插入正常
- [ ] 坐标修正正常

### 6. 目录 (TOC)
- [ ] 打开目录面板
- [ ] 目录结构正确
- [ ] 点击目录跳转正常
- [ ] 目录折叠/展开正常

### 7. 搜索功能
- [ ] 打开搜索面板
- [ ] 输入搜索关键词
- [ ] 搜索结果高亮显示
- [ ] 跳转到搜索结果正常

---

## Performance Tests (性能测试)

### 1. 大文件测试
- [ ] 打开 10MB 文件 < 100ms
- [ ] 打开 50MB 文件 < 200ms
- [ ] 滚动流畅，无卡顿

### 2. 快速输入测试
- [ ] 快速输入 200 字符/秒
- [ ] 屏幕刷新正常，无闪烁失控
- [ ] 输入延迟 < 50ms

### 3. 手写延迟测试
- [ ] 手写笔延迟 < 16ms
- [ ] 笔迹与触摸点一致
- [ ] 坐标修正有效

---

## E-ink Refresh Tests (E-ink 刷新测试)

### 1. 局部刷新
- [ ] 打字输入时局部刷新正常
- [ ] 滚动时局部刷新正常
- [ ] 光标移动时局部刷新正常

### 2. 全局刷新
- [ ] 分屏切换时全局刷新一次
- [ ] 模式切换时全局刷新一次
- [ ] 目录展开时全局刷新一次
- [ ] 普通操作不应触发全局刷新

### 3. 刷新质量
- [ ] 无残影
- [ ] 无闪烁过度
- [ ] 刷新区域正确

---

## Memory Tests (内存测试)

### 1. 内存占用
- [ ] 空闲时内存 < 100MB
- [ ] 打开 50MB 文件后内存 < 300MB
- [ ] 长时间使用后内存稳定

### 2. 内存清理
- [ ] 后台运行时触发内存清理
- [ ] 清理后 UI 不白屏
- [ ] 清理后功能正常

### 3. 低内存测试
- [ ] 后台运行多个应用
- [ ] 触发 onTrimMemory 回调
- [ ] 应用不崩溃

---

## Crash Tests (崩溃测试)

### 1. 边界测试
- [ ] 打空文件

- [ ] 快速滚动

### 2. 异常测试

- [ ] 磁盘空间不足
