### 安装步骤

1. 下载安装包
2. 解压到指定目录
   - Windows: `C:\Program Files\NomadMark`
   - macOS: `/Applications/NomadMark`
   - Linux: `/opt/nomadmark`
3. 配置环境变量（如果需要）
4. 启动应用程序

### 支持的平台

- **移动端**
  - Android 5.0+
  - iOS 12.0+

- **桌面端**
  - Windows 10+
  - macOS 10.14+
  - Linux (Ubuntu 18.04+, Fedora 28+)

请检查以下内容：

1. ✓ 无序列表的符号是否正确显示
2. ✓ 有序列表的编号是否正确对齐

5. ✓ 列表中的格式化（粗体、代码、链接）是否正常

7. ✓ 中英文混排的列表显示是否正常

9. ✓ 空列表项的处理是否正确
