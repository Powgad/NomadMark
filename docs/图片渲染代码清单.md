# 图片渲染相关代码清单

## 核心文件

### 1. 图片处理器
**文件**: [ImageProcessor.kt](platforms/android/app/src/main/java/com/editor/nomadmark/image/ImageProcessor.kt)

**功能**:
- 预读取图片尺寸（`readImageDimensions()`）
- 计算采样率（`calculateInSampleSize()`）
- 采样解码图片（`decodeSampledBitmapFromUri()`）
- JPEG 压缩保存（`saveBitmapAsJpeg()`）
- 主要处理方法（`processAndSaveImage()`）

**关键常量**:
```kotlin
const val MAX_SCREEN_WIDTH = 1872
const val JPEG_QUALITY = 85
const val IMAGE_DIR = "markdown_images"
```

### 2. Coil 图片加载器配置
**文件**: [SupernoteImageLoader.kt](platforms/android/app/src/main/java/com/editor/nomadmark/image/SupernoteImageLoader.kt)

**功能**:
- 初始化 Coil 图片加载器（`init()`）
- 创建 Markwon ImageRequest（`createMarkwonRequest()`）
- 清理缓存（`clearMemoryCache()`, `clearDiskCache()`）

**配置参数**:
```kotlin
crossfade = false           // 禁用动画
allowHardware = false       // 禁用硬件位图
allowRgb565 = true          // 使用 RGB565
precision = INEXACT         // 允许近似尺寸
```

### 3. 图片管理器
**文件**: [MarkdownImageManager.kt](platforms/android/app/src/main/java/com/editor/nomadmark/image/MarkdownImageManager.kt)

**功能**:
- 提取 Markdown 中的图片路径（`extractImagePaths()`）
- 清理未使用的图片（`cleanUnusedImages()`）
- 导出图片（`exportImage()`）

### 4. 调试工具
**文件**: [ImageTestUtils.kt](platforms/android/app/src/main/java/com/editor/nomadmark/image/ImageTestUtils.kt)

**功能**:
- 检查图片目录状态（`checkImageDirectory()`）

**图片处理流程**:
```kotlin
private fun insertImageFromUri(uri: android.net.Uri) {
    // 1. 请求持久化读取权限
    // 2. 使用 ImageProcessor 处理图片
    val relativePath = imageProcessor.processAndSaveImage(uri)
    // 3. 获取完整路径
    val fullPath = imageProcessor.getFullPath(relativePath)
    // 4. 插入 file:// 路径到 Markdown
    insertImage("file://$fullPath")
}
```

## 图片处理流程

```
用户点击图片按钮
    ↓
showImageInsertDialog() - 显示对话框
    ↓
┌─────────────┬─────────────┐
│             │             │
选择本地图片    输入网络图片
│             │             │
openImagePicker()  showImageUrlDialog()
│             │             │
获得 content://   用户输入 URL
│             │             │
insertImageFromUri()  insertImage(url)
│             │             │
ImageProcessor.processAndSaveImage()
    - readImageDimensions()
    - calculateInSampleSize()
    - decodeSampledBitmapFromUri()
    - saveBitmapAsJpeg()
    ↓                │
返回相对路径         │
markdown_images/xxx.jpg
    ↓                │
getFullPath()        │
    ↓                │
insertImage("file:///.../markdown_images/xxx.jpg")
    ↓                │
生成 Markdown: ![图片](file://...)
```

## 已知问题

### 1. Markwon ImagesPlugin 默认配置
- Markwon 4.6.2 的 `ImagesPlugin.create()` 使用默认配置
- 可能不支持 file:// 协议（需要额外配置）
- 默认使用 Glide 而非 Coil

### 2. Coil 初始化
- Coil 已在 Application 中初始化
- 但 Markwon 可能没有使用 Coil 来加载图片

### 3. 图片目录访问权限
- 图片保存在 app-private 目录
- Markwon 可能需要特殊配置才能访问该目录

## 下一步

如果图片不显示，需要：
1. 配置 Markwon 使用 Coil 加载图片
2. 添加 file:// 协议支持
3. 或使用其他方式访问本地图片
